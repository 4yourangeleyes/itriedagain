## Understanding Your Contract System

Your contract system has **3 main components**:

### 1. **Contract Template Blocks** (21 templates)
Pre-configured templates in the database with:
- Contract type (Service Agreement, NDA, etc.)
- Pre-selected clauses appropriate for that contract type
- Ready to use immediately

### 2. **Clause Library** (18 professional clauses)
Reusable SA-compliant clauses covering:
- **Project Definition**: Scope, Deliverables, Acceptance
- **Financial**: Payment Terms, Change Requests
- **Legal**: IP Ownership, Confidentiality, Liability, Termination
- **Service Quality**: Warranty, Support, Testing
- **Project Management**: Communication, Data Protection, Force Majeure
- **Contract Management**: Dispute Resolution, Governing Law, Entire Agreement

### 3. **Clause Manager** (Interactive UI)
Browse, search, and customize clauses for your contracts.

## How to Create Contracts: 3 Methods

### Method 1: Quick Contract (Recommended for Beginners)

**Best for:** Standard contracts with pre-selected professional clauses

1. **Click the Plus button** in header
2. **Select or add a client**
3. **Choose "Contract"**
4. **Select contract type:**
   - **SERVICE_AGREEMENT** ‚Üí General professional services (most common)
   - **PROJECT_CONTRACT** ‚Üí Specific project with deliverables
   - **RETAINER** ‚Üí Ongoing monthly services
   - **NDA** ‚Üí Non-disclosure only
   - **MAINTENANCE** ‚Üí Website/software maintenance
   - **EMPLOYMENT** ‚Üí Employment contracts
   - **CONSULTING** ‚Üí Consulting engagements
   - **LICENSE** ‚Üí Software/IP licensing
   - **SHAREHOLDER** ‚Üí Shareholder agreements
   - **PARTNERSHIP** ‚Üí Business partnerships
   - **SUPPORT** ‚Üí Technical support agreements

5. Contract is created with **automatically selected clauses** appropriate for that type

**Example - Creating a Service Agreement:**
```
Plus ‚Üí Select Client "ABC Company" ‚Üí Contract ‚Üí SERVICE_AGREEMENT
```

**Result:** Contract with these clauses:
- Scope of Work
- Payment Terms and Schedule
- Intellectual Property Rights
- Confidentiality and Non-Disclosure
- Warranties and Guarantees
- Liability and Indemnification
- Termination and Cancellation
- Change Requests and Amendments
- Client Acceptance and Sign-off
- Communication and Reporting
- Testing and Quality Assurance
- Dispute Resolution and Arbitration
- Governing Law and Jurisdiction
- Entire Agreement and Amendments

### Method 2: Custom Contract with Clause Manager

**Best for:** Contracts that need specific clause combinations

1. Create a basic contract using Method 1
2. On canvas, **open Clause Manager** (should be a button on contract view)
3. **Browse the Clause Library:**
   - View by category (Project Definition, Financial, Legal, etc.)
   - See which clauses are "required" (recommended for all contracts)
4. **Add/Remove clauses:**
   - Add new clauses from library
   - Remove clauses you don't need
   - Reorder clauses with up/down arrows
5. **Edit clause content:**
   - Click "Edit" on any clause
   - Customize text while maintaining legal integrity
6. **Save your contract**

**Example - Creating a Custom NDA with Extra Protection:**
```
1. Create NDA contract (has basic clauses)
2. Open Clause Manager
3. Add "Data Protection and GDPR Compliance" clause
4. Add "Breach Notification" clause
5. Reorder to put most important clauses first
6. Save
```

### Method 3: Create from Template Block

1. Go to **Canvas** or **Chat** screen

## Understanding Contract Clauses

### Required Clauses (Recommended for ALL contracts)
These 8 clauses form the foundation of any professional contract:

1. **Scope of Work** - What you're delivering
2. **Payment Terms** - How and when you get paid
3. **IP Ownership** - Who owns the work
4. **Confidentiality** - Protecting sensitive information
5. **Warranty** - Quality guarantees
6. **Liability** - Limitation of legal responsibility
7. **Termination** - How to end the contract
8. **Entire Agreement** - Legal boilerplate

### Optional Clauses (Add as needed)

**For Project-Based Work:**
- Change Requests and Amendments
- Client Acceptance and Sign-off
- Testing and Quality Assurance
- Project Deliverables

**For Ongoing Services:**
- Support and Maintenance
- Communication and Reporting
- Service Level Agreements (SLA)

**For Protection:**
- Data Protection and GDPR Compliance
- Breach Notification Requirements
- Force Majeure
- Dispute Resolution

**For Complex Agreements:**
- Sub-contracting Rights
- Equipment and Resources
- Travel and Expenses
- Non-compete and Non-solicitation

## Troubleshooting

### "My contract has no clauses!"

**Solution 1: Re-run setup script**
```bash
npm run setup-northcell
```
This reloads all templates with clauses.

**Solution 2: Add clauses manually**
1. Open contract on canvas
2. Click "Open Clause Manager"
3. Browse library and add required clauses
4. Start with the 8 required clauses
5. Add optional clauses as needed

### "I don't see Clause Manager button"

Check that:
- You're in a CONTRACT document (not invoice)
- You're on the Canvas screen
- Look for a button like "Manage Clauses" or "Edit Clauses"

### "Clause content is too generic"

**Solution: Customize each clause**
1. Open Clause Manager
2. Click "Edit" on each clause
3. Replace placeholder text:
   - "[Company Name]" ‚Üí "Northcell Studios"
   - "[Client Name]" ‚Üí Actual client name
   - "[Project Description]" ‚Üí Specific project details
   - "[Amount]" ‚Üí Actual fee
   - "[Timeline]" ‚Üí Actual dates
